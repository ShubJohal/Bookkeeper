# Bookkeeper
Full-stack book tracking app built with Node.js, Express, EJS, and PostgreSQL. Supports adding, editing, and deleting books, ratings, notes, live search, sorting, and filtering. Integrates Open Library API for book covers with manual upload fallback and dynamic UI updates.

## Features

- Add, edit, and delete books  
- Rate books (1–5 stars)  
- Add personal notes  
- Live search, sorting, and filtering  
- Book cover integration via Open Library API  
- Manual image upload fallback (Multer)  
- Dynamic UI updates (no page reload feel)  
- Clean date formatting

## Tech Stack
- Frontend: EJS, HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: PostgreSQL
- Other Tools: Multer, Axios

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/bookkeeper.git
cd bookkeeper
``` 

## 2. Install dependencies

```bash
npm install

```
## 3. Setup PostgreSQL databse

``` bash
CREATE TABLE books (
  id INT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  title TEXT NOT NULL,
  author TEXT NOT NULL,
  date_read DATE,
  rating INT NOT NULL CHECK (rating BETWEEN 1 AND 5),
  notes TEXT,
  book_cover TEXT,
  created_at TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP
);

```

 ## 4. Configure database
 Update your db.js file with your PostgreSQL credentials.

 ## 5. Run the app

 ``` bash
npm start 

```

App runs on:

``` bash
http://localhost:3000

```

## How It Works
- Books are stored in PostgreSQL
- Covers are fetched using Open Library API
- If no cover is found, users can upload one manually
- Frontend dynamically updates the book list using JavaScript
